# StockX Interview

## Description

Create a github repository for the `true shoe size`.

## Installation in Development

```bash
$ npm install
```

## Running the app

This makes the assumption that you're at the root of the folder.

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
